        Description

        Per discussion in wireframes there is a request to update the policy regarding start dates of service on web orders.  It was decided the reason behind the delay is no longer valid in today's processes (immediate payment collected).  Please update so that the service start date (regular and plus for all products) for new licenses all have a start date that matches the order date.  

        This is due to concerns from a customer regarding immediate access to service benefits which was escalated from CS to policy deciders.  

        Joy
